# InformationTheoryAssignments
ELL714- Assignments of Information Theory Sem7

- Joint Typicality Coding: Encoding using random codebook and decoding using joint typicality rules. If an output is joinly typical with only one codeword in the codebook, the decoded output would be the message corresponding to that codeword.

- Polar Coding: Polar coding for a Binary Erasure Channel. Decoded using Successive Cancellation Decoding.
